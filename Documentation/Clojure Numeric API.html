1. Arithmetic Functions

Function Description Examples

(+x1x2…xn)(+x1x2…xn)

Returns x1+x2+⋯+xnx1+x2+⋯+xn. Returns 0 if no arguments are specified. Throws an ArithmeticException if an overflow is produced when using arguments of type Long.

 
(+ 1 2 3 4 5)
=> 15
(+)
=> 0
(+ Long/MAX_VALUE 1)
=> ArithmeticException

(+'x1x2…xn)(+'x1x2…xn)

Returns x1+x2+⋯+xnx1+x2+⋯+xn. Returns 0 if no arguments are specified. Converts result automatically into BigInt if an overflow is produced when using arguments of type Long.

 
(+' 1 2 3 4 5)
=> 15
(+')
=> 0
(+' Long/MAX_VALUE 1)
=> 9223372036854775808N

(𝚒𝚗𝚌x)(incx)

Returns x+1x+1. Throws an ArithmeticException if an overflow is produced when its argument is of type Long.

 
(inc 5)
=> 6
(inc -5)
=> -4
(inc Long/MAX_VALUE)
=> ArithmeticException

(𝚒𝚗𝚌'x)(inc'x)

Returns x+1x+1. Converts result automatically into BigInt if an overflow is produced when its argument is of type Long.

 
(inc' 5)
=> 6
(inc' -5)
=> -4
(inc' Long/MAX_VALUE)
=> 9223372036854775808N

(-x1x2…xn)(-x1x2…xn)

Returns x1−x2−⋯−xnx1−x2−⋯−xn. Returns −x1−x1 if only one argument is provided. Throws an ArithmeticException if an overflow is produced when using arguments of type Long.

 
(- 10 1 2 3)
=> 4
(- 10)
=> -10
(- Long/MIN_VALUE 1)
=> ArithmeticException

(-'x1x2…xn)(-'x1x2…xn)

Returns x1−x2−⋯−xnx1−x2−⋯−xn. Returns −x1−x1 if only one argument is provided. Converts result automatically into BigInt if an overflow is produced when using arguments of type Long.

 
(-' 10 1 2 3)
=> 4
(-' 10)
=> -10
(-' Long/MIN_VALUE 1)
=> -9223372036854775809N

(𝚍𝚎𝚌x)(decx)

Returns x−1x−1. Throws an ArithmeticException if an overflow is produced when its argument is of type Long.

 
(dec 5)
=> 4
(dec -5)
=> -6
(dec Long/MIN_VALUE)
=> ArithmeticException

(𝚍𝚎𝚌'x)(dec'x)

Returns x−1x−1. Converts result automatically into BigInt if an overflow is produced when its argument is of type Long.

 
(dec' 5)
=> 4
(dec' -5)
=> -6
(dec' Long/MIN_VALUE)
=> -9223372036854775809N

(*x1x2…xn)(*x1x2…xn)

Returns x1×x2×⋯×xnx1×x2×⋯×xn. Returns 1 if no arguments are specified. Throws an ArithmeticException if an overflow is produced when using arguments of type Long.

 
(* 1 2 3 4 5)
=> 120
(*)
=> 1
(* Long/MAX_VALUE 2)
=> ArithmeticException

(*'x1x2…xn)(*'x1x2…xn)

Returns x1×x2×⋯×xnx1×x2×⋯×xn. Returns 1 if no arguments are specified. Converts result automatically into BigInt if an overflow is produced when using arguments of type Long.

 
(*' 1 2 3 4 5)
=> 120
(*')
=> 1
(*' Long/MAX_VALUE 2)
=> 18446744073709551614N

(/x1x2…xn)(/x1x2…xn)

Returns x1÷x2÷⋯÷xnx1÷x2÷⋯÷xn. Returns 1x11x1 if only one argument is provided.

 
(/ 20 2 5)
=> 2
(/ 1 2)
=> 1/2
(/ 10.0)
=> 0.1

(𝚚𝚞𝚘𝚝xy)(quotxy)

Returns the integer quotient of dividing xx by yy.

 
(quot 20 3)
=> 6
(quot -20 3)
=> -6
(quot -20 -3)
=> 6

(𝚛𝚎𝚖xy)(remxy)

Returns the remainder of dividing xx by yy.

If aa is (𝚚𝚞𝚘𝚝xy)(quotxy), and bb is (𝚛𝚎𝚖xy)(remxy), then it holds true that: (a×y)+b=x(a×y)+b=x 
(rem 20 3)
=> 2
(rem -20 3)
=> -2
(rem 20 -3)
=> 2
(rem -20 -3)
=> -2

(𝚖𝚘𝚍xy)(modxy)

Returns xmodyxmody (modulus of xx and yy). This is basically equivalent to:

(- x(* (𝙼𝚊𝚝𝚑/𝚏𝚕𝚘𝚘𝚛 (/ xy)) y))(- x(* (Math/floor (/ xy)) y))

(𝚖𝚘𝚍xy)(modxy) and (𝚛𝚎𝚖xy)(remxy) produce the same results when xx and yy have the same sign, but this is not necessarily true otherwise. 
(mod 20 3)
=> 2
(mod -20 3)
=> 1
(mod 20 -3)
=> -1
(mod -20 -3)
=> -2

2. Numerical Predicates

Function Description Examples

(𝚣𝚎𝚛𝚘?x)(zero?x)

Returns true if xx is zero, otherwise returns false.

 
(zero? 0)
=> true
(zero? 42)
=> false
(zero? 0.0)
=> true

(𝚙𝚘𝚜?x)(pos?x)

Returns true if xx is a number greater than zero, otherwise returns false.

 
(pos? 42)
=> true
(pos? -10)
=> false
(pos? 0)
=> false

(𝚗𝚎𝚐?x)(neg?x)

Returns true if xx is a number less than zero, otherwise returns false.

 
(neg? 42)
=> false
(neg? -10)
=> true
(neg? 0)
=> false

(𝚎𝚟𝚎𝚗?x)(even?x)

Returns true if xx is an even number, otherwise returns false. Throws an IllegalArgumentException if xx is not an integer.

 
(even? 10)
=> true
(even? 15)
=> false
(even? 2.0)
=> IllegalArgumentException

(𝚘𝚍𝚍?x)(odd?x)

Returns true if xx is an odd number, otherwise returns false. Throws an IllegalArgumentException if xx is not an integer.

 
(odd? 10)
=> false
(odd? 15)
=> true
(odd? 2.0)
=> IllegalArgumentException

3. Comparison Functions

Function Description Examples

(=x1x2…xn)(=x1x2…xn)

Returns true if x1=x2=⋯=xnx1=x2=⋯=xn, otherwise returns false. Two numbers x1x1 and x2x2 are equal only if both have the same type and the same value.

This function also works with non-numeric values. 
(= (* 3 1)
   (+ 2 1)
   (/ 12 4))
=> true
(= (* 2 3) (+ 2 3))
=> false
(= 1 1.0)
=> false

(==x1x2…xn)(==x1x2…xn)

Returns true if all numbers x1,x2,…,xnx1,x2,…,xn have equivalent type-independent values, otherwise returns false.

 
(== (* 3 1)
    (+ 2 1)
    (/ 12 4))
=> true
(== (* 2 3) (+ 2 3))
=> false
(== 1 1.0)
=> true

(𝚗𝚘𝚝=x1x2…xn)(not=x1x2…xn)

Same as (𝚗𝚘𝚝(=x1x2…xn))(not(=x1x2…xn)).

This function also works with non-numeric values. 
(not= (* 3 1)
      (+ 2 1)
      (/ 12 4))
=> false
(not= (* 2 3) (+ 2 3))
=> true
(not= 1 1.0)
=> true

(<x1x2…xn)(<x1x2…xn)

Returns true if x1<x2<⋯<xnx1<x2<⋯<xn, otherwise returns false.

 
(< 1 2 3 4 5)
=> true
(< 1 2 3 3 4)
=> false
(< 10 4)
=> false

(<=x1x2…xn)(<=x1x2…xn)

Returns true if x1⩽x2⩽⋯⩽xnx1⩽x2⩽⋯⩽xn, otherwise returns false.

 
(<= 1 2 3 4 5)
=> true
(<= 1 2 3 3 4)
=> true
(<= 10 4)
=> false

(>x1x2…xn)(>x1x2…xn)

Returns true if x1>x2>⋯>xnx1>x2>⋯>xn, otherwise returns false.

 
(> 5 4 3 2 1)
=> true
(> 5 4 4 3 2)
=> false
(> 4 10)
=> false

(>=x1x2…xn)(>=x1x2…xn)

Returns true if x1⩾x2⩾⋯⩾xnx1⩾x2⩾⋯⩾xn, otherwise returns false.

 
(>= 5 4 3 2 1)
=> true
(>= 5 4 4 3 2)
=> true
(>= 4 10)
=> false

4. Random Numbers

Function Description Examples

(𝚛𝚊𝚗𝚍)(rand)

(𝚛𝚊𝚗𝚍n)(randn)

Returns a random floating point number rr, where 0⩽r<n0⩽r<n. If nn is omitted it defaults to 1.

 
(rand)
=> 0.3473911330833691
(rand 100)
=> 87.43000630622653

(𝚛𝚊𝚗𝚍-𝚒𝚗𝚝n)(rand-intn)

Returns a random integer number rr, where 0⩽r<n0⩽r<n.

 
(rand-int 10)
=> 4
(rand-int 100)
=> 63

5. Numeric Tower Functions

These are math functions that deal intelligently with the various types in Clojure’s numeric tower.

To use these functions, you need to import them by doing something like this (replace abs with the function names you want to use):

(require '[clojure.math.numeric-tower :refer [abs]])
Function Description Examples

(𝚊𝚋𝚜x)(absx)

Returns |x||x| (absolute value of xx).

 
(abs -5)
=> 5
(abs 12N)
=> 12N
(abs -1.2)
=> 1.2

(𝚜𝚚𝚛𝚝x)(sqrtx)

Returns x√x (square root of xx), exact if possible.

 
(sqrt 25)
=> 5
(sqrt 2)
=> 1.4142135623730951
(sqrt -1)
=> ##NaN

(𝚎𝚡𝚙𝚝xy)(exptxy)

Returns xyxy (xx raised to the power yy).

 
(expt 2 10)
=> 1024
(expt 2 1/2)
=> 1.4142135623730951
(expt 2 -3)
=> 1/8

(𝚛𝚘𝚞𝚗𝚍x)(roundx)

Returns ⌊x⌉⌊x⌉ (xx rounded to the nearest integer). Rounds up for values exactly in between two integers. The result is always an integer.

 
(round 1.2)
=> 1
(round 1.5M)
=> 2N
(round -2.9)
=> -3

(𝚌𝚎𝚒𝚕x)(ceilx)

Returns ⌈x⌉⌈x⌉ (least integer greater than or equal to xx, or in other words, xx rounded towards +∞+∞). If xx is an exact number, returns an integer, otherwise a double.

 
(ceil 1.2)
=> 2.0
(ceil 1.5M)
=> 2N
(ceil -2.9)
=> -2.0
(ceil 1/8)
=> 1N

(𝚏𝚕𝚘𝚘𝚛x)(floorx)

Returns ⌊x⌋⌊x⌋ (greatest integer less than or equal to xx, or in other words, xx rounded towards −∞−∞). If xx is an exact number, returns an integer, otherwise a double.

 
(floor 1.2)
=> 1.0
(floor 1.5M)
=> 1N
(floor -2.9)
=> -3.0
(floor 1/8)
=> 0N

(𝚐𝚌𝚍xy)(gcdxy)

Returns the greatest common divisor of xx and yy. Arguments must be integers.

 
(gcd 8 12)
=> 4

(𝚕𝚌𝚖xy)(lcmxy)

Returns the least common multiple of xx and yy. Arguments must be integers.

 
(lcm 8 12)
=> 24
